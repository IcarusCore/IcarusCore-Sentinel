Our Mission

To democratize access to cybersecurity threat intelligence by aggregating, analyzing, and presenting threat data from multiple authoritative sources in an intuitive, actionable format.

In today's rapidly evolving threat landscape, security teams need immediate access to the latest threat intelligence to protect their organizations. Our platform bridges the gap between raw threat data and actionable security insights.

Key Features

Real-time Updates

Automatic data refresh from multiple threat intelligence sources with configurable intervals.

MITRE ATT&CK Integration

Native integration with the MITRE ATT&CK framework for standardized threat categorization.

Advanced Filtering

Powerful search and filtering capabilities with support for multiple criteria and saved searches.

Analytics & Insights

Visual analytics and trend analysis to identify patterns and emerging threats.

Quick Start Guide

Installation Steps
1. Clone the Repository
git clone https://github.com/your-repo/threat-intel-dashboard.git
cd threat-intel-dashboard
2. Create Virtual Environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
3. Install Dependencies
pip install -r requirements.txt
4. Configure Environment (Optional)
export OTX_API_KEY="your-otx-api-key"
export FLASK_DEBUG=True
5. Run the Application
python app.py
Note: The application will be available at http://localhost:5000 once started.

Configuration Options

Basic Configuration

Essential settings for getting started:

  • DEBUG - Enable debug mode for development
  • SECRET_KEY - Flask application secret key
  • DATA_DIR - Directory for storing threat data
  • ITEMS_PER_PAGE - Pagination settings
API Configuration

API keys for external data sources:

  • OTX_API_KEY - AlienVault OTX API access
  • SHODAN_API_KEY - Shodan API access (future)
  • REFRESH_INTERVALS - Data update frequencies
  • RSS_FEEDS - Custom RSS feed sources
